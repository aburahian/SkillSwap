# SkillSwap

## Project Purpose

SkillSwap is a web application that allows users to discover, book, and offer skill-based learning sessions. It connects learners with skilled providers across various categories, making learning accessible and convenient.

## Live URL

https://coruscating-kringle-38aa49.netlify.app/,
https://skillswap-7nl.pages.dev/

## Key Features

- **User Authentication:** Register and login securely with email/password.
- **Profile Management:** Update user display name and profile photo.
- **Protected Routes:** Certain pages (like booking a session or profile) are only accessible to logged-in users.
- **Skill Listings:** Browse skills with details such as name, provider, rating, price, and available slots.
- **Skill Details Page:** View full information about a skill, including description and provider info.
- **Book Session:** Fill a simple form to book a session with success notifications.
- **Responsive Design:** Works smoothly on desktop and mobile devices.
- **Toast Notifications:** Alerts for successful actions (registration, booking, etc.).
- **Client-Side Validation:** Password strength and form input checks for better UX.
